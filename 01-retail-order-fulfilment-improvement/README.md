# Retail Order Fulfilment Improvement Project

## Project Overview
This project is an end-to-end Business Analysis and analytics case study for a simulated omnichannel retailer experiencing fulfilment, stock-out and OTIF performance issues.

The project demonstrates how a Business Analyst can connect business problem definition, stakeholder needs, process improvement, KPI design, Excel/data analysis, Power BI dashboarding, Agile delivery and AI-assisted decision support.

## Business Problem
The retailer had reasonably strong forecasting performance, but fulfilment remained inconsistent. The analysis identified that low OTIF performance was mainly linked to stock-out risk, supplier under-delivery, delivery variability and weak exception management rather than forecast accuracy alone.

## Key Insight
Forecast accuracy alone does not guarantee fulfilment success. The biggest improvement opportunity is in supplier reliability, replenishment control and exception-based management.
