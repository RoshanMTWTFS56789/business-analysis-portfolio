# business-analysis-portfolio
Business Analysis, IT Business Analysis, Power BI, Excel, SQL and consulting portfolio projects.
End-to-end Business Analysis project for a simulated omnichannel retailer facing fulfilment, stock-out and OTIF performance issues.  
Methods used: stakeholder analysis, KPI design, As-Is/To-Be process mapping, Agile user stories, Excel KPI analysis, Power BI dashboarding and AI-assisted decision support.
